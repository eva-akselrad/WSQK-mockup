# WSQK 102.5 FM - The Voice of Hawkins

A retro-styled web mockup of WSQK FM, the fictional radio station from Hawkins, Indiana (circa 1983).

## Features

- 80s-inspired CRT and VHS visual effects
- Animated cassette tape player interface
- News ticker with Hawkins-themed messages
- Responsive design with neon aesthetics

## Getting Started

Simply open `index.html` in your browser to view the mockup.

## Deployment

This site is automatically deployed to GitHub Pages when changes are pushed to the main branch.
